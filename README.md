# Machine Learning vs Deep Learning: Comparative Study

## Abstract

Machine learning (ML) and deep learning (DL) have become fundamental technologies in modern artificial intelligence systems. While both approaches aim to extract patterns from data and enable predictive analytics, they differ significantly in architecture, feature engineering, performance, and application domains. This study presents a comparative case analysis of ML and DL, synthesizing foundational concepts from prior literature and integrating recent research contributions. The paper examines differences in data processing, model construction, evaluation metrics, and practical deployment challenges. The findings highlight that while deep learning excels in high-dimensional and unstructured data environments, traditional machine learning methods remain competitive in scenarios with limited data and higher interpretability requirements. The study concludes with implications for selecting appropriate models in real-world applications.

## Authors
Luke Freire, Nathan Kibbe, Indy Park

## Purpose
This project demonstrates understanding of key differences between machine learning and deep learning, including performance trade-offs, model complexity, and real-world applications.
